Habilidades

Cascading Style Sheets (Folhas de estilo em cascata) é um mecanismo para adicionar estilos a uma página Web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags style. Estudei esse mecanismo inicialmente na plataforma da Alura. Depois, na plataforma da Origamid.

HyperText Markup Language (Linguagem de Marcação de Hiperexto) - Linguagem que permite criar programas que trabalham com textos e imagens numa mesma tela simultaneamente. Tomei conhecimento desta linguagem há muito tempo, mas só nos últimos me dediquei ao seu estudo.

Uma linguagem de script, compacta e baseada em objeto, desenvolvida pela Netscape Communications como uma linguagem de macros para servidores. É utilizada na editoração para a Web (www), pois os autores da Web podem incorporar instruções simples de programação semelhantes às da linguagem Java no texto HTML de suas páginas. Minha experiência com esta linguagem, em comparação com as anteriores, é mais recente. Fiz cursos na Web com o prof. Guanabara, na Alura e, recentemente, na Origamid.

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML. Estudei essa linguagem recentemente em um curso do prof. Gustavo Guanabara.

Portifólio

Ler Livros

Ler Livros é uma programa para guardar uma lista de livros lidos. Ele nasceu da necessidade de saber quantos livros em conseguiria ler em uma ano. Tudo começou quando o meu médico me perguntou quantos livros eu lia por mês. Como eu não tinha a resposta, resolvi anotar num caderno o nome do autor, o título, a quantidade de páginas e a categoria do livro.
